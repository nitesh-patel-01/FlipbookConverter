How it works

From flat file to flippable book in three steps.

  1. 01

    Upload

    Drop a PDF or a set of images. We validate by magic bytes — no executables sneak in.

  2. 02

    Render & optimise

    We rasterise pages at print-friendly DPI and compress them with Sharp for fast loads.

  3. 03

    Preview & download

    Flip pages live, then export a portable HTML ZIP you can host on any static server.

Features

FAQ

Questions, answered.

What is the max file size?

By default 50 MB per upload and up to 300 pages per PDF. You can raise both by editing environment variables.

Do you store my files?

Uploads and generated assets are removed automatically after two hours. Nothing is shared with third parties.

Can I host the exported flipbook?

Yes — the ZIP is a self-contained HTML bundle. Upload the folder to any static host: Netlify, Vercel, GitHub Pages, S3, Render Static, etc.

Does it work offline after download?

Almost. The exported HTML loads jQuery and turn.js from CDN by default. For fully offline use, drop both files next to flipbook.html and update the script tags.
